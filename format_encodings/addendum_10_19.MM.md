1. Convert 10# 234.75 -> 2# 0 1110 1010 . 110
   1. Convert 10# 234 -> 2# 0 1110 1010
   
      ```
      number = 234
   
            234 / 2 -> 117 , 0
            117 / 2 -> 58 , 1
            58 / 2 -> 29, 0
            29 / 2 -> 14, 1
            14 / 2 -> 7, 0
            7 / 2 -> 3, 1
            3 / 2 -> 1, 1
            1 / 2 -> 0, 1
            0 / 2 -> 0, 0
   
      anwser: 0 1110 1010
      ```

   1. Convert 10# .75 -> 2# 0.
      ```
      number:   75
      max:    1,00

      75 * 2 -> 1,50   -> 1
      50 * 2 -> 1,00   -> 1
      00 * 2 -> 0,00   -> 0

      answer = 110
      ```
